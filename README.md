# 🎓 Student Performance Analytics & Segmentation Dashboard

## 📌 Project Overview

This project focuses on analyzing student academic, behavioural, and lifestyle data to uncover meaningful performance patterns and segment students into distinct groups based on their learning habits and engagement levels. The objective is to move beyond basic exploratory data analysis and build an intelligent analytics system that helps in understanding how different factors such as study hours, attendance, sleep patterns, and internet usage influence academic performance.

The project follows a complete data science workflow starting from data cleaning and exploratory data analysis to feature engineering and student clustering. Instead of building a prediction model initially, the focus is on deriving actionable insights and presenting them through an interactive dashboard for better academic decision-making and performance monitoring.

---

## 🚀 Main Objectives

* Perform structured data preprocessing and exploratory data analysis on student data
* Identify behavioural and academic patterns affecting performance
* Create engineered features such as performance index, engagement score, and lifestyle indicators
* Segment students into meaningful clusters using unsupervised learning techniques
* Build an interactive dashboard for visualization and insight exploration
* Provide a foundation for future machine learning model integration

---

## ⚙️ Project Workflow

1. **Data Cleaning & Preprocessing**

   * Handling missing values
   * Removing duplicates
   * Detecting and managing outliers
   * Formatting and validating dataset

2. **Exploratory Data Analysis (EDA)**

   * Univariate analysis of categorical and numerical features
   * Distribution visualization using histograms and boxplots
   * Relationship discovery using correlation heatmaps and scatter plots

3. **Feature Engineering**

   * Creating behavioural indicators such as performance index and engagement score
   * Combining lifestyle and academic features for deeper analysis

4. **Student Segmentation (Clustering)**

   * Applying clustering techniques to group students based on performance and behaviour
   * Identifying high performers, average learners, and at-risk students

5. **Dashboard Development**

   * Visualizing student clusters and performance trends
   * Creating filters for behavioural attributes
   * Enabling interactive academic insight exploration
